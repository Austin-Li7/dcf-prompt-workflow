# Sample Selection

## Chosen Tickers

- `MSFT`
- `AAPL`
- `CRM`

## Why These Three

- `MSFT`: richest existing artifact set, best baseline for review
- `AAPL`: second large-cap company to check whether Step 1 validation behavior generalizes
- `CRM`: intended enterprise software / SaaS contrast sample

## Current Constraint

Only MSFT and AAPL currently have pipeline outputs available in the workspace. CRM remains part of the test plan, but not yet part of the evidence base.
